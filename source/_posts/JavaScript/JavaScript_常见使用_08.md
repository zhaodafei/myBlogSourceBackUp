### 指定范围内随机数

```js
// 定义一个 dateRandom()函数，原理是 随机数和最大值减最小值的差相乘 最后再加上最小值。
function dateRandom(min, max) {
  return Math.floor(Math.random() * (max - min)) + min
}
```
