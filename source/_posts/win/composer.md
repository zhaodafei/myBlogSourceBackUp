### 查看composer 全局设置

```
composer config -gl

composer 镜像源配置文件:
C:\Users\Administrator\AppData\Roaming\Composer\config.json
```

### 修改镜像源

```
国内镜像
composer config -g repo.packagist composer https://packagist.phpcomposer.com
国际镜像
composer config -g repo.packagist composer https://packagist.org
```
