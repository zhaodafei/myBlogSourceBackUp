### 基础配置

```wiki
#恢复原始配置
在这个目录 C:\Users\fei\AppData\Roaming\Qoder\User 中有个文件 settings.json,删除即可

#设置编辑器颜色
头像 / 编辑器设置 / 主题(搜索) / 常用设置( 选择: Qoder Light )

#设置编辑器字体
头像 / 编辑器设置 / 字体(搜索) / 文本编辑器/字体 / 字体大小(16) 
## webstorm 中
File / Setting / Editor / Font / size(16)
```
