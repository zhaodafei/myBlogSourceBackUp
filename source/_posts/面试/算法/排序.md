### 冒泡排序

``` php
<?php
// 冒泡排序
function bubbleOrder($arr)
{
    $count = count($arr);
    $temp = 0;
    // 外层控制排序次数
    for ($i = 0; $i < $count - 1; $i++) {
        // 内层控制每轮比较次数
        for ($j = 0; $j < $count - 1 - $i; $j++) {
            if ($arr[$j] > $arr[$j + 1]) {
                $temp = $arr[$j];
                $arr[$j] = $arr[$j + 1];
                $arr[$j + 1] = $temp;
            }   // print_r($arr);  echo "<br>";
        }
    }
    return $arr;
}

$arr = array(5, 2, 7, 6, 9, 3);
$res = bubbleOrder($arr);
var_dump($res);
```
