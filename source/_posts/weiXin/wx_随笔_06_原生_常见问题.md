### 上传文件格式

```js
wx.chooseImage({
  success (res) {
    const tempFilePaths = res.tempFilePaths
    wx.uploadFile({
      url: 'https://example.weixin.qq.com/upload', //仅为示例，非真实的接口地址
      filePath: tempFilePaths[0],
      name: 'file',
      formData: {
        'user': 'test'
      },
      success (res){
        const data = res.data
        //do something
      }
    })
  }
})
```

这个功能中隐藏的问题, 在安卓手机中会把`png`格式的图片转为`jpg`格式, 后端通过二进制流获取图片格式和源文件格式对比就会出现异常

1. [wx.chooseMedia 选择原图在安卓中，png 格式的图片临时路径后缀名会是 .jpg](https://developers.weixin.qq.com/community/develop/doc/000ee89879099884be342fe5866800)
2. [wx.chooseImage IOS手机和安卓（非原图） 会被强行转为JPG](https://developers.weixin.qq.com/community/develop/doc/000ee8b7d708f847a0a64bb6d5b000?highLine=wx.uploadFile%2520jpg)
3. [wx.uploadFile接口上传jpg图片, content-type类型错误](https://developers.weixin.qq.com/community/develop/doc/becd925aa7a3b4b1c303cccb9656b86d?highLine=wx.uploadFile%2520jpg)
4. [小程序打开相册，上传png图片，就会生成jpg的临时路径，这个问题怎么解决](https://developers.weixin.qq.com/community/develop/doc/0000481a2605c806f3d62875a56c00?highLine=wx.uploadFile%2520jpg)
