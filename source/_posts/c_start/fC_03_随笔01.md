### 数组

数组初始化

```c
int a[10] = {0, 1, 2, 3, 4};

char str[ ] = {'a', 'b', 'c', 'd', 'e' };

```

C语言除了在定义数组变量时用初值列表对数组整体赋值以 外，无法再对数组变量进行整体赋值。

如果再想赋值,可以使用循环逐一赋值

```c
// demo:  删除数组中下标为2的元素(本质是修改)
// 解释:第五个元素的内存还在,数组是一个连续的空间, 总长度不会变,只是把值改变了
// 初始化时内存空间已经申请下来了，就不会改变了
int i, b[5] = {99, 60, 75, 86, 92};
int n = 5;
for (i = 2; i < n - 1; i++) {
    b[i] = b[i + 1];
}
n = n - 1;

for ( i = 0; i < n; i++) {
    printf("%d ", b[i]);
}

printf("%d ", b[4]);
```
