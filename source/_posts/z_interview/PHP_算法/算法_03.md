### 01). 输出 `9*9` 乘法口诀

``` php
for ($x = 1; $x < 10; $x++) {
    for ($y = 1; $y <= $x; $y++) {
        echo $y . "*" . $x . "=" . $y * $x;
        echo "&nbsp&nbsp&nbsp";
    }
    echo "<br>";
}
```

### 02) 猴子吃桃

```php
//题目：猴子吃桃问题：猴子第一天摘下若干个桃子，当即吃了一半，还不瘾，又多吃了一个    
//第二天早上又将剩下的桃子吃掉一半，又多吃了一个。
//以后每天早上都吃了前一天剩下 的一半零一个。到第10天早上想再吃时，见只剩下一个桃子了。求第一天共摘了多少。 

$x = 1; //最后一天的桃子个数
for ($i = 2; $i <= 10; $i++) { //天数循环
    $x = ($x + 1) * 2;
}
echo "猴子第1天条摘了" . $x . "个桃子";
```
